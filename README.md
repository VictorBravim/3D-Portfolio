Portfolio 3D - Spline

## 📋 Requirements

- Node.js
- npm (Node Package Manager)
- framer-motion
- Spline3D
- react-icons

## 🔧 Installation

1. Clone this repository:
   
```
git clone https://github.com/VictorBravim/3D-portfolio.git
```

2. Navigate to the project directory:
   
```
cd 3D-portfolio
```

3. Install dependencies:
   
```
npm install
```
